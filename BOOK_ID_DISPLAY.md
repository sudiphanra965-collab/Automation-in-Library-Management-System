### Database Schema
```sql

  id INTEGER PRIMARY KEY AUTOINCREMENT,  -- âœ… Auto-generated!
  title TEXT,
  author TEXT,
  isbn TEXT,
  category TEXT,
  description TEXT,
  image TEXT,
  available INTEGER DEFAULT 1
```

### ID Assignment Process
1. Admin adds new book (no ID specified)
2. SQLite database auto-generates next available ID
3. Book saved with ID: 17, 18, 19, etc.
4. ID permanently associated with that book
5. ID visible in admin panel, book details, etc.
